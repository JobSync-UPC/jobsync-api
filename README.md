## Getting Started
### Installation
1. Clone the repo
   ```sh
   git clone https://github.com/JobSync-UPC/jobsync-api
   ```
3. Run the project in CLI or run in http configuration with IDEs
   ```sh
   mvn spring-boot:run
   ```
